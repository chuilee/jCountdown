jCountdown - jQuery Countdown Plugin Demos

Advanced Template Demo

Creating multiple instances of jCountdown. In v1.5.0 this is made easier with the use of a new option dataAttr.

Simply add a date to each element using the HTML5 data-* attribute: 

<p fix"></p>

Then pass the data name to the plugin: 

$(".time").countdown({
	date: "december 28, 2016 00:00",
	timeWrapElement: 'span',
	textWrapElement: 'span',
	timeWrapClass: 'cd-time',
	textWrapClass: 'cd-text',
	timeSeparator: '',
	spaceCharacter: ' '
});
